# Projekt-Brief: Agency Dashboard

**Eingereicht:** 2026-03-24
**Vom Board:** Oliver Hees

## Was soll gebaut werden?
Management-Dashboard für die KI-Agentur — besser als Paperclip.
GitHub ist die einzige Datenquelle.

## Kernfunktionen (MVP)
- [ ] Org-Chart live (Agents, Status, Budget)
- [ ] Projekt-Übersicht (Progress, ETA, Risiko)
- [ ] Token-Kosten pro Agent in Echtzeit
- [ ] Board Escalations sofort sichtbar
- [ ] Daily Standup Feed
- [ ] Agent spawnen direkt aus UI

## Tech
- Next.js 15, TypeScript, Tailwind CSS
- GitHub API (keine eigene DB)
- GitHub OAuth (nur Oliver)
- Hetzner + Coolify
- Signal Red #E53E3E + Void Black
- Playwright Visual Tests
