**Deadline:** 2026-04-04

## Beschreibung

React-Komponente `OrgChart` bauen, die Agent-Profildaten als Karten-Grid darstellt. Farbschema: Signal Red für Highlights und Fehler, Void Black als Hintergrund. Die Komponente nutzt React Query für Daten-Fetching und zeigt Skeleton-Loader während des Ladens.

## Technische Details

- Client Component (`'use client'`) mit React Query Hook: `useQuery({ queryKey: ['org-chart'], ... })` laut ADR-001 und ADR-002
- `staleTime: 5 * 60 * 1000` laut ADR-002
- Server Component für initiales Laden (Prefetch in `app/page.tsx`) laut ADR-001
- Skeleton-Loader während Loading-State (kein Spinner laut ADR-002)
- Farbcodierung Status laut US-1: Grün = active, Grau = idle/pending, Rot = error/offline
- Signal Red bei Agents über 30% Budget-Anteil laut US-2
- Grid-Layout: bis zu 10 Agents ohne Scrollen (Viewport 1440x900) laut US-1 AC-4
- Agent-Karte enthält: Name, Rolle, Status-Badge, Budget-Anteil %, `commit_prefix`
- Inline-Error mit "Retry"-Button bei Fetch-Fehler laut ADR-002
- Tailwind CSS für Styling, keine externe Component Library
